Tittlen på artikkelen

Når man skal lage en nettside, er det viktig at man strukturere siden ordentlig med en layout som både er fin og lett å manøvrere. Dette er ikke alltid en simpel oppgave, men ved hjelp av CSS, og noen veldig gode layout metoder, så kan du gjøre det ganske så lett å strukturer siden din.

CSS Flexbox

Ved hjelp av Flexbox, kan du lett forandre egenskapene til hovedelementen og child elementen. Ved hjelp av flex, kan du velge en retning du vil at child elementen skal dra ved hjelp av flex-direction, har du lyst på en liste som går nedover eller oppover, kan du gjøre det lett ved hjelp av flex. Du kan også bruke flex-wrap, som vil sørge for child elementene vil være på flere linjer, istedenfor å skvise seg i en linje. Du kan også bruke kommandoer som justify-content, for å bestemme hvor child elementene skal starte og om de skal ha mellomrom. Du kan også bestemme hvordan child elementen skal se ut, du kan fortelle at noen skal være større enn andre eller mindre. Du kan bestemme rekkefølgen til elementene, og om en av elementen skal plasseres annerledes fra de andre. CSS flex box er en veldig god måte og strukutrer seg på.

Hvis du vil lære mer om flexbox, kan du sjekke denne siden.

CSS Grid

CSS Grid er en annen metode for å ordne med layout. Det er mye likt mellom grid og flexbox, men den største fordelene med grid over flexbox er at, mens en flexbox kan enten jobbe med rader eller kolonner, kan grid jobbe med begge samtidig. En grid vil lage soner som du kan gi navn til, for så og sette elementer inn i den sonen, på denne måten kan du lett få struktur på siden din. Du kan også gi de forskjellige sonen større areal enna andre, hvis det skal mer tekst eller et bilde i den. Du kan bestemme hvor stort mellomrom det skal være mellom sonene du lager, både mellom kollonen og raden. Du kan også bestemme om elementene i sonen skal være på starten(øverst), slutten (nederst), sentrert eller strekke seg over hele sonen. Du kan også forandre på child-elementene inne i selve sonen, bestemme om de skal strekke seg over 3 kolloner og 2 rader for eksempel.

Hvis du vil lære mer om grid, kan du sjekke denne siden.
